# Prep Doc Context

Use this context for start-of-week or status-prep requests.

- Gather durable status from active worker sessions, workflow handoffs, GitHub or
  Linear state, monitor findings, and recent lifecycle events.
- Lead with blockers, failed CI, PRs needing review, and decisions required from
  humans.
- Summarize quiet repos briefly.
- If assembling the prep doc requires multiple repo checks or long-running
  commands, the director launches an `adhoc` engineer worker to produce it.
